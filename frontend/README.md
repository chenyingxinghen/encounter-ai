# 青春伴行 - 前端应用

基于React的青春伴行平台前端用户界面。

## 技术栈

- **React 18.2** - UI框架
- **React Router 6** - 路由管理
- **Axios** - HTTP客户端
- **Vite** - 构建工具

## 页面结构

### 1. 登录页面 (LoginPage)
- 用户邮箱和密码登录
- 表单验证
- 错误提示
- 跳转到注册页面

### 2. 注册页面 (RegisterPage)
- 用户基本信息填写（用户名、邮箱、密码、学校、专业、年级）
- 表单验证
- 注册成功后跳转到测评页面

### 3. 测评问卷页面 (AssessmentPage)
- **步骤1**: MBTI人格测试（4个问题）
- **步骤2**: 大五人格评估（5个问题，1-5分评分）
- **步骤3**: 兴趣标签选择（学业、职业、兴趣爱好）
- **步骤4**: 场景选择（考研自习室、职业咨询室、心理树洞、兴趣社群）
- 进度条显示
- 完成后跳转到匹配页面

### 4. 匹配结果展示页面 (MatchingPage)
- 场景切换（考研自习室、职业咨询室、心理树洞、兴趣社群）
- 显示匹配用户列表（最多10个）
- 每个匹配卡片显示：
  - 用户基本信息
  - 匹配度分数
  - MBTI类型
  - 兴趣标签
  - 匹配理由
- 开始对话按钮

### 5. 聊天界面 (ChatPage)
- 左侧对话列表
- 右侧聊天区域
- 实时消息显示
- AI助手建议（当检测到沉默时）
- 消息发送功能
- 自动滚动到最新消息

### 6. 个人中心页面 (ProfilePage)
- 基本信息展示和编辑
- 人格特质展示（MBTI、大五人格）
- 兴趣标签展示
- 关注场景展示
- 社交统计（对话总数、匹配次数、平均对话质量）

### 7. 成长报告页面 (ReportPage)
- 报告类型切换（周报、月报、年报）
- 核心数据展示（对话总数、消息总数、平均对话质量、社交能力）
- 情绪健康评分和趋势
- 成长亮点列表
- 改进建议列表
- 数据可视化
- 报告下载功能
- 报告分享功能

### 8. 设置页面 (SettingsPage)
- AI助手开关
- 隐私设置（匿名模式、数据收集授权）
- 通知设置
- 隐私政策查看
- 数据导出
- 账号删除

## 组件

## 运行项目

### 安装依赖
```bash
cd frontend
npm install
```

### 开发模式
```bash
npm run dev
```
访问 http://localhost:3000

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## API集成

前端通过Axios与后端API通信，所有API请求都通过 `/api` 前缀代理到后端服务器（默认 http://localhost:8000）。

主要API端点：
- `/api/auth/login` - 用户登录
- `/api/auth/register` - 用户注册
- `/api/users/profile/assessment` - 提交测评结果
- `/api/matching/find` - 获取匹配结果
- `/api/conversations/*` - 对话相关接口
- `/api/users/profile/*` - 用户画像接口
- `/api/reports/*` - 成长报告接口
- `/api/users/settings/*` - 用户设置接口

1. 确保后端API服务运行在 http://localhost:8000
2. 所有需要认证的页面都有路由保护
3. 用户状态通过React state管理（可以升级为Context或Redux）
4. 聊天页面每3秒自动刷新消息
5. 所有表单都有基本的验证
